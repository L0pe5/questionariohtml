Introdução

-> Clone o projeto no seu computador, crie uma branch com o nome "Respostas". A ideia da tarefa é que você construa o html para responder o conteudo de Git e HTML.

1 - O que é um elemento HTML?

Um elemento HTML é o conjunto completo fotmado por uma tag de abertura, o conteúdo e a tag de fechamento.

2 - Qual a diferença entre head e body?

A diferença entre o head e o body é que as informações contidas no head não são exibidas na página da web, e tudo que é inserido no body faz parte do conteúdo visual da página.

3 - O que é um atributo HTML?

O atributo é uma informação extra inserida dentro de uma tag que serve para configurar ou ajustar seu comportamento.

4 - Como é feito o comentário em HTML?

O comentário em HTML é feito com: , por exemplo: .

5 - O que é uma tag?

A tag é a estrutura do HTML, elas são utilizadas para indicar a função de cada parte do conteúdo.

6 - O que é Git?

O Git é uma ferramenta que guarda o histórico das alterações/versões do código.

7 - O que é GitHub?

É uma rede social onde os usuários podem armazenar e compartilhar seus projetos.

8 - O que é uma branch?

A branch é o espaço de trabalho. Ela é uma ramificação do projeto, podendo ser criada para criar ou testar algo sem alterar o código principal

9 - O que é commit?

O commit é a gravação de um comentário no código que informa quais alterações/atualizações foram realizadas.

10 - Para que serve o Git?

Como o Git registra as versões do código ele pode ser utilizado para facilitar o trabalho em equipe. Assim mais de uma pessoa pode realizar alterações sem sobrescrever o arquivo.

11 - O que são títulos no HTML?

São tags usadas para definir a importância dos textos dentro da página.

12 - Qual a diferença entre "h1" e "h6"?

Todas são tags utilizadas em títulos. Além delas possuirem uma hieraquia, outro ponto que diferencia elas é o tamanho, sendo que h1 é a maior delas, utilizada como título principal, e a h6 é a menor.

13 - Para que serve a tag "p"?

A tag p serve para criação de um parágrafo.

14 - O que é um formulário HTML?

O formulário é uma seção dentro do HTML utilizada para recolher dados inseridos pelo usuário.

15 - Para que serve a tag "form"?

Serve para agrupar todos os campos de entrada de dados dentro do formulário, denine o destino e o método de envio desses dados.
